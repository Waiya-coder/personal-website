Walmart Sales Forecasting Tool

Walmart Sales Predictor is a machine learning-powered tool designed to forecast weekly sales across Walmart stores. The application integrates data from various sources, including markdowns, store size, and holiday indicators, to predict store-level sales trends. Key features include:

    Integration of advanced regression and time-series models (Random Forest and LSTM) for accurate sales predictions.
    Evaluation of models using RMSE and MAE metrics, with performance optimizations through hyperparameter tuning.
    Insights into key predictors like markdowns and store size to inform inventory and marketing strategies.
  • Data Science
  • Machine Learning
